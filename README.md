# sebovy-bankovni-ucty

Oficiální repozitář Sebových bankovních účtů a karet.

⚠️ Přístup pouze pro:
- Seba
- Sebu
- Sebíka
- Policii (automaticky)

## Obsah
- cards/ – karty (velmi citlivé)
- pins/ – PINy (ještě citlivější)
- accounts/ – účty (nejcitlivější)
- policie/ – automatické logování přístupů

## Jak použít
1. Otevři soubor
2. Uvědom si, že to byl špatný nápad
3. Zavři soubor
